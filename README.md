# Python 資料分析實務

本專案為 Python 程式設計與資料分析實作，涵蓋資料處理、數值運算與資料視覺化分析。

透過完整實作流程，建立 Python 在資料分析領域的核心能力。

---

## 📌 專案內容

本專案依資料分析流程分為三大部分：

### 🐍 Python 程式基礎
涵蓋變數、資料型態、流程控制、函式、物件導向與例外處理等基礎程式能力。

---

### 📊 資料處理與分析
使用 NumPy 與 Pandas 進行資料處理、數值運算與基礎統計分析。

---

### 📈 資料視覺化
透過 Matplotlib 與 SciPy 進行資料視覺化與統計分析，輔助資料理解與呈現。

---

## 🎯 專案目標

建立 Python 資料分析能力，包含：

- 程式設計與邏輯能力
- 資料處理與分析能力
- 數據運算與統計能力
- 基礎資料視覺化能力

---

## 📁 專案說明

- 本專案為 Python 基礎至資料分析完整實作整理
- 著重於資料分析流程與實務應用能力
- 不包含外部 BI 工具整合（如 Tableau）
